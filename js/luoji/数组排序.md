## 数组排序

```javascript{.line-numbers}
    const  arr = [23, 15, 6, 46, 8, 163, 13, 51, 85, 3, 1385, 1, 0, 35, 4, 8546];
```
#### 1. 冒泡排序
```javascript{.line-numbers}
    for (let i = 0; i < arr.length; i++) {
        for (let j = 0; j <= i; j++) {
            if (arr[i] < arr[j]) {
                var tamp = arr[i];
                arr[i] = arr[j];
                arr[j] = tamp;
            };
        };
    };
    // [0, 1, 3, 4, 6, 8, 13, 15, 23, 35, 46, 51, 85, 163, 1385, 8546]
```
#### 2. 使用 sort 方法
```javascript{.line-numbers}
    arr.sort((a,b)=>{
        return a-b;
    })
    // [0, 1, 3, 4, 6, 8, 13, 15, 23, 35, 46, 51, 85, 163, 1385, 8546]
```
